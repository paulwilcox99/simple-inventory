================================================================================
HOW TO RUN YOUR LLM INVENTORY AGENT
================================================================================

Expected directory layout:
  widget-sim/                  <- manufacturing simulator
  ├── venv/
  ├── databases/
  ├── run_simulation.py
  └── simple-inventory/        <- this agent (must be inside widget-sim/)
      └── venv/

================================================================================
FIRST TIME SETUP
================================================================================

Step 1: Install simulator dependencies (from widget-sim/):
  python3 -m venv venv
  ./venv/bin/pip install -r requirements.txt
  ./venv/bin/python create_sim.py

Step 2: Install agent dependencies (from simple-inventory/):
  python3 -m venv venv
  ./venv/bin/pip install -r requirements.txt

Step 3: Set your OpenAI API key (from simple-inventory/):
  cp .env.example .env
  # edit .env and add: OPENAI_API_KEY=sk-...

================================================================================
OPTION 1: QUICK TEST (EASIEST - START HERE)
================================================================================

Step 1: Open terminal and navigate:
  cd simple-inventory

Step 2: Run a single inventory check:
  ./venv/bin/python llm_inventory_agent.py --once

That's it! The agent will:
  - Read inventory from database
  - Analyze with AI
  - Show you the decision
  - Execute restocking if needed

Time: ~10-30 seconds
Cost: ~$0.03 (GPT-4) or ~$0.005 (GPT-3.5)

================================================================================
OPTION 2: USE HELPER SCRIPTS
================================================================================

Even easier! From simple-inventory/ just run:
  ./run_once.sh

Or for cheaper/faster AI:
  ./venv/bin/python llm_inventory_agent.py --once --model gpt-3.5-turbo

================================================================================
OPTION 3: RUN WITH SIMULATION (ADVANCED)
================================================================================

You need TWO terminal windows:

Terminal 1 - Start Simulation (from simple-inventory/):
  ./start_simulation.sh

Terminal 2 - Start Agent (from simple-inventory/):
  ./start_agent.sh

The agent will monitor the simulation and make decisions automatically!
The simulation pauses 5 seconds between days so the agent has time to act.

Duration: ~30-60 seconds for 30-day simulation (with 5s delay)
Cost: ~$0.10-0.50 depending on how many decisions

================================================================================
OPTION 4: TEST SETUP (NO API CALLS)
================================================================================

Want to verify everything works without spending money?
  ./venv/bin/python test_agent.py

This checks:
  ✓ API key configured
  ✓ Databases accessible
  ✓ Can connect to OpenAI
  ✓ LLM responds correctly

================================================================================
WHAT YOU'LL SEE
================================================================================

The agent prints:
  🤖 Initialization info
  📊 Inventory data summary
  🧠 "Consulting LLM..." (this is where AI thinks)
  💭 LLM's decision (RESTOCK or NO_ACTION)
  📝 Reasoning (why the AI made this choice)
  📋 Parts to restock with quantities
  📦 Execution (database updates)
  💰 Total cost
  ✅ Success confirmation

================================================================================
TIPS
================================================================================

• First time? Use: ./run_once.sh
• Want it cheap? Add: --model gpt-3.5-turbo
• Want detailed output? Just run it - already verbose!
• Something wrong? Run: ./venv/bin/python test_agent.py
